插入排序

插入排序（Insertion Sort）。

插入排序思想

算法的基本思想是把n个带排序元素看成一个有序表和一个无序表。开始时有序表只包含1个元素，无需表包含n-1个元素，从无序表中取出第一个元素，将它插入到有序表中的适当位置，使之成为新的有序表，重复n-1次完成排序。

插入排序的分析

  • 平均时间复杂度为O(n 2)

  • 空间复杂度为O(1)

  • 最差时间复杂度为O(n 2)

    最差的情况为逆序

  • 最好时间复杂度为O(n)

    最好的情况为顺序

  • 插入排序稳定

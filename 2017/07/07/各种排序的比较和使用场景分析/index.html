各种排序的比较和使用场景分析

各种排序的复杂度和稳定性

排序算法 最差时间复杂度 平均时间复杂度 最好时间复杂度 空间复杂度 空间复杂度
插入排序 O(n 2 ) O(n 2 ) O(n) O(1) 稳定
冒泡排序 O(n 2 ) O(n 2 ) O(n) O(1) 稳定
选择排序 O(n 2 ) O(n 2 ) O(n 2 ) O(1) 不稳定
希尔排序 - O(n)-O(n 2 )之间 - O(1) 不稳定
堆排序 O(nlogn) O(nlogn) O(nlogn) O(1) 不稳定
归并排序 O(nlogn) O(nlogn) O(nlogn) O(n) 稳定
快速排序 O(n 2 ) O(nlogn) O(nlogn) O(log n) 不稳定
基数排序 O(dn) O(dn) O(dn) O(n) 稳定

使用场景

平均时间复杂度为O(n 2 )的算法

  • 插入排序、冒泡排序、选择排序平均的时间复杂度都为O(n 2
  • 它们适合数据规模较小的情况，，其中插入和冒泡适合那些基本有序的序列，在基本有序的情况下趋近于O(n)。

平均时间复杂度为O(nlongn) 的算法

  • 快排，归并，堆排序的时间复杂度都为O(nlongn)
  • 归并排序稳定，快排和堆排序不稳定。
  • 快速排序最好，但在基本有序的情况下会变成冒泡排序。
  • 堆排序非常适合在很多数中找出前几个最小的数这类的问题。

其他

  • 希尔排序的时间复杂度介于O(n)-O(n 2 )之间 ，它达不到线性，也不至于到二次。希尔排序对增量的标准没有较为满意的答案，增量对性能会有影响。
  • 基数排序是桶排序的变形，可用多趟桶排序进行排序，在一些情况下比其他排序算法好。
